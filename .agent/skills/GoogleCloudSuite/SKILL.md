# GoogleCloudSuite

Skill para la interacción centralizada con servicios de Google Cloud (Vision y Places).

## Instrucciones

Usar esta skill para llamadas al backend que requieran análisis de imagen (Vision) o búsqueda de lugares (Places API) asegurando el manejo de cuotas y API Keys.

### Herramientas

- `analyze_image_vision`: Llama a Vision API para detección de etiquetas.
- `search_places_keyword`: Realiza búsquedas de POIs filtradas por las etiquetas de Vision.
